## Submission Notes

- Ensure NAP (Name, Address, Phone) is EXACTLY consistent across all listings
- Use the same business name format everywhere
- Use the exact same phone number format
- Use the same website URL format (include or exclude www consistently)
- Upload the same logo/photos to all platforms
- Select the same categories where possible
- Keep hours of operation consistent
